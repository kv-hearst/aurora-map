About PolarMap.js

This example demonstrates PolarMap.js, which extends Leaflet to support polar projections. The map can display both Arctic (North Pole) and Antarctic (South Pole) views using proper polar stereographic projections.

Features:

  • EPSG:3995 - Arctic Polar Stereographic projection
  • EPSG:3031 - Antarctic Polar Stereographic projection
  • Full Leaflet compatibility for markers, popups, and overlays
  • Proper coordinate transformation for polar regions
